# Price Action Training（K线认知训练系统 · Trading Gym）

一个**纯前端、零依赖、可离线运行**的 Price Action 认知训练系统。它不是选股 / 量化 / 模拟盘工具，而是一套刻意练习（Deliberate Practice）引擎——只训练你"看图 → 做决策"的能力，评分看**决策质量**而非盈亏。

> 双击 `index.html` 即可开始训练。无需安装、无需联网、无需构建。

---

## ✨ 核心原则

1. **禁止未来函数（No Future Function）**
   你永远只能看到"当前光标"之前的数据。图表右侧约 20% 被锁定为 `🔒 未来数据已隐藏`，只能逐步"揭示下一根 K 线 / 下一天 / 下一周"。这强迫你像真实盘中一样做决策。

2. **训练优先于盈亏（Training > Profit）**
   评分基于决策过程，而不是赚了多少钱。一笔正确的交易即使亏了也高分；一笔错误的交易即使蒙对了也扣分。

3. **过程与结果分离（Process ≠ Outcome）**
   每期训练都给出四维评分与弱点标签，帮助你把注意力放在"我为什么这样决策"上。

训练"真相"用**事后回看（hindsight）**标注：
- 突破真假 → 看后续 8 个交易日价格行为
- 情绪周期 → 看 next-5 日累计收益
- 板块轮动 → 看次日领涨板块变化
- K线 / 量能 / 趋势结构 → 确定性分类函数

## 🚀 快速开始

### 方式一：直接打开（最简单）
双击 `index.html`。浏览器即可运行（首屏会从 `data/market_data.js` 加载真实 A 股历史日线）。

> ⚠️ 若用 `file://` 直接打开，部分浏览器会禁用本地存储（localStorage），此时训练记录仅当前会话有效、刷新即清空。顶部会有提示。
> 想**永久保存记录**，请用本地服务器打开（方式二）。

### 方式二：本地服务器（推荐，记录可持久化）
```bash
cd <项目目录>
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## 📊 数据

默认数据集 `data/market_data.js`（20 只 A 股、2018–2026 真实日线）**不纳入版本库**（体积约 4 MB，且可复现）。

重新生成数据：
```bash
pip install requests   # 如未安装
python3 fetch_data.py  # 新浪财经为主、腾讯财经兜底，生成 data/market_data.js
```

`fetch_data.py` 会从新浪 / 腾讯拉取 20 只标的最近约 2000 根日线，写入 `window.MARKET_DATA = {...}` 供前端直接 `<script>` 引入。

## 🧪 测试

无头集成测试（无需浏览器，用 DOM 桩驱动全流程）：
```bash
node smoke_test.js
```
会依次：启动 → 模块1盲测全流程 → 渲染全部模块 → 各模块提交（含完整评分路径）→ 成长中心 → 校验产出记录无 NaN / 脏分。

## 🏗️ 架构

```
index.html            入口（顺序加载 data → core → 各模块 → TG.start()）
core.js               TG 命名空间：持久化、等级阶梯、模块导航、drawCandles 自绘K线引擎
styles.css           深色主题样式（涨红跌绿，A股惯例）
m1_blind.js          模块1 K线盲测
m2_breakout.js       模块2 真/假突破
m3_pa.js             模块3 Price Action
m4_market.js         模块4 市场结构（并构建全市场日期索引）
m5_sector.js         模块5 板块轮动（复用 m4 的日期索引）
m6_position.js       模块6 仓位管理
m7_risk.js           模块7 风险控制
m8_center.js         模块8/9/10 成长中心（日志 + 错误分析 + 等级）
data/market_data.js  真实行情数据集（由 fetch_data.py 生成）
fetch_data.py        数据拉取脚本（新浪/腾讯）
smoke_test.js        无头集成测试
```

技术上**没有任何构建步骤、没有任何第三方 CDN 依赖**：K 线用 Canvas 自绘，数据直接以 JS 全局变量提供，持久化用 localStorage（file:// 受限时自动内存兜底）。
